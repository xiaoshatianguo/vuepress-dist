# 关于webstorm的使用指南

# 安装

yarn add -D vuepress

# 运行

  1. 建一个文件夹写下markdown.md文件
  2. 运行命令：docs表示md文件所在的上一层目录
npx vuepress dev docs

# 配置

  1. 目录结构 ├─ docs │ ├─ README.md │ └─ .vuepress │ └─ config.js └─ package.json
  2. 配置文件
module.exports = {
  title: 'Hello VuePress',
  description: 'Just playing around'
}
